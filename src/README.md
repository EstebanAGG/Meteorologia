# Procesador de datos Meteorológicos
## Descripción
Desarrolla un procesador concurrente de grandes datos 
meteorológicos usando Kotlin, aplicando corrutinas, 
concurrencia y programación funcional.
## Requisitos
1. Desarrolla una aplicación en Kotlin que procese grandes 
volúmenes de datos meteorológicos utilizando un enfoque 
funcional y concurrente.
2. Utiliza corrutinas (async, await, launch) para manejar la 
concurrencia de manera eficiente.
3. Aplica funciones de orden superior y procesamiento funcional
de listas para manipular los datos.
4. Asegura la inmutabilidad y la consistencia de datos en un
entorno concurrente.
5. Los datos meteorológicos para el procesamiento se deben
simular mediante un generador que produzca al menos 10,000 registros con atributos como temperatura, humedad y velocidad del viento.
6. Implementa una función principal que actúe como punto de 
entrada y gestione la ejecución global de las tareas asíncronas.
7. Divide el código en archivos lógicos separados: 
uno para la generación de datos, otro para el procesamiento 
y otro para las funciones auxiliares.
